# Better Colony Automation (BCA) - 殖民地自动化改良

由于罪恶的P社在4.0更新后一直没修复自动托管的bug，导致每到中后期的游戏体验极差，最后还得是自己来解决P社的bug

## 快速开始

1. **全局设置**：在【政策】中设置**自动建造矿物预留**。
2. **开启自动化**：在行星界面开启【行星自动化】以及对相关自动化选项进行配置。
3. **精准微调**：点击【行星决策】，可额外开启**允许为仓库建造额外区划及特化**或**理想城候选**（需要可建设理想城才会显示）等。

## 总览

### 建造功能

#### 自动建造逻辑
- **按需建造**：仅在岗位不足、且星球不存在建设中建筑的情况下才进行建造。可以通过行星决策进行无视岗位需求的提前建造
- **矿物预留**：增加了国家级的**矿物预留政策**。你可以设置矿物阈值，若国家矿物低于该值，自动化将暂停建设，避免造成经济崩溃。

#### 增加自动化选项
- **医疗**：支持基础医疗建筑、基因组研究所及分子焕活所的建设。
- **文化丰碑**：自动建造凝聚力产出建筑。
- **资源仓储**：自动建造仓库。
- **人口管理细化**：（非机械帝国）可独立管理机器人组装厂的建设。
- 移除了避免财政赤字的选项，若不手动关闭会经常卡住建筑建设。

#### 新增规划类型
新增物理/社会/工程专项科研星球，以及对应的“科研+贸易”、“工业+贸易”等混合型专精。
注：当前版本的混合类型类型是半成品，主要用于理想城/环世界/蜂巢星球/机械星球，且贸易混合类型不会尝试建造贸易建筑，计划后继版本优化。

#### 建筑建设
优先建造辅助性建筑，如诊所、克隆舱、文化丰碑、生产部等、以提升星球整体效能，主产出建筑在辅助建筑建造完成后再进行建设。
本模组对原版的自动建造逻辑进行了大量改良，包括适配各种特殊星球类型，稀有资源建筑的建造逻辑也得到了修复。

- **仓库建设**：默认只建造一个仓库。可以通过行星决策来建造更多仓库，其还会自动建造一产区块以容纳仓库建筑。
- **分子焕活所**：默认只建造一个焕活所。可以通过行星决策来解除限制。

#### 区划建设
- **优先级优化**：模组将在一产专精星球上优先建设一产区划，避免因优先级冲突导致的一产区划无法建设的问题。
- **理想城/环世界/蜂巢星球/机械星球**：将优先建造未特化分区各一个，以增加建筑槽。

### 拆除功能

#### 建筑拆除
- **职能不符拆除**：当建筑槽位已满且当前建筑不符合星球当前的专精时，模组会自动将其移除。例如：在科研球上有工厂且没地方造实验室时，工厂会被拆除。
- **人口组装腾挪**：当星球彻底填满（无住房、无岗位、无剩余空位）时，人口组装建筑会被自动拆除，为更高级的产出建筑腾出空间。可以通过政策关闭此功能。

#### 区划拆除
- **一产区划**：通过开启“理想城项目候选”决策，将在无空余区划时移除一级资源（发电/采矿/农业）区划，直到满足项目启动条件。
- **城市区划**：行星拥有冗余住房，且为了建设特定的其它区划，会移除过剩的城市区划。

#### 区划特化拆除
当玩家手动更改行星专精时（例如从科研球改为贸易球），会自动移除星球上不适宜的旧特化（如研究特化等）。研究特化之间的转型（如社会专精转物理专精）不会触发移除

### 其他

- **理想城候选决策**：增加了“理想城预备选项”的行星策略。将自动拆除一产区划，并在满足建设条件时提示玩家是否正式开启项目。
- **去城市化决策**：提供了一个快速移除所有建筑、区划的决策（其实主要是用于模组测试），你并不需要使用这个决策来进行行星转型。
- **政策控制**：可以在政策菜单中全局开启/关闭各等级的自动拆除功能。

## 兼容性

- 本模组对的**建筑拆除**功能对原版建筑进行了修改，可能会与修改原版建筑的模组产生冲突。建议在使用本模组时避免加载其他修改原版建筑的模组。如果你能容忍没有建筑拆除功能，可以在加载顺序上将本模组放在修改原版建筑的模组之前。
- 修改ai行为的模组可能会与本模组产生冲突，建议将本模组放在加载顺序的最后。
- 本模组只支持原版建筑和区划的自动托管，暂不支持任何来自其他模组的建筑和区划。
